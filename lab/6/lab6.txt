Директивы ассемблера masm.

1. proc
Директива для определения функций.

fun_name proc [c | stdcall] [uses esi edi ebx,] [arg1:DWORD[, arg2:DWORD]]
	local i:dword
	local buf[10]:byte


	ret
fun_name endp

Если в функции используются локальные переменные или аргументы автоматически создаются стандартный пролог и эпилог.


2. Проверка условий

.if условие_1
	...
.elseif условие_2

	...
	
.elseif условие_n
	...
.else
	...
.endif

Возможные условия:
op1 ==(!=, <, >, <=, >=) op2
op1 & op2     test op1, op2
!op1
op1 && op2
op1 || op2
CARRY?		; выставлен флаг CF
OVERFLOW?	; выставлен флаг OF
PARITY?		; выставлен флаг PF
SIGN?		; выставлен флаг SF
ZERO?		; выставлен флаг ZF


3. Циклы

.while условие
	.continue	; переход к следующей итерации
	.break		; выход из цикла
	.continue .if условие	; переход к следующей итерации при выполнении условия
	.break .if условие		; выход из цикла при выполнении условия
	...
.endw
